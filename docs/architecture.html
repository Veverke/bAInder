System Architecture

bAInder is a Chrome / Edge Manifest V3 browser extension that captures, organises, and searches conversations from AI chat platforms (ChatGPT, Claude, Gemini, Copilot).

Five architectural layers — each with a clearly bounded responsibility. The data flows top-down: the Content Script extracts conversations from AI pages and posts them to the Background Service Worker, which persists them via the Shared Library. The Side Panel and Reader read from the same shared storage and render the saved data.
